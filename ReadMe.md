# ANDI

**ANDI** is a Python library for controlling the **Digilent Analog Discovery 2** or **Digilent Analog Discovery 3** instruments.

> *Python library for controlling the Analog Discovery 2 or 3 module from Digilent.*

---

## Installation

Install from PyPI with `pip`:

```bash
pip install andi-py
```

---

## Requirements

### Third-party software (mandatory)

**Digilent WaveForms** must be installed before using ANDI.

- Download and install WaveForms from Digilent:  
  https://digilent.com/shop/software/digilent-waveforms/

### Python

ANDI has been developed for **Python 3**.

### Recommended packages

- `numpy`

---

## Notes on device compatibility

ANDI was developed for the **Analog Discovery 2**, which is discontinued.

So far, there seem to be **no major problems** using **Analog Discovery 3**.
